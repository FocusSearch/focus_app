### 优化
1、优化内容生成流程

| 级别 | 品牌角色 | 提及量 | 适用条件 |
|------|---------|-------|---------|
| L1 纯行业 | 不提及 | 0 | 无企业信息 |
| L2 类别引用 | 类别中的一个实例 | 2-3 次 | 有基础信息 |
| L3 方法锚定 | 方法论的实施示例 | 3-5 次 | 有完整画像 |
| L4 案例嵌入 | 具体实施证据 | 4-6 次 | 有客户案例/数据 |
| L5 专题说明 | 分析对象 | 按类型 | 落地页/案例研究 |

| 模板 | 适用场景 | 长度 | 品牌上限 |
|------|---------|------|---------|
| 深度文章 | 日常博客、行业分析 | 1500-3000 字 | L3 方法锚定 |
| 要点总结 | 梳理类、清单类 | 800-1500 字 | L2 类别引用 |
| 概念解读 | 概念科普、趋势分析 | 1500-3000 字 | L3 方法锚定 |
| 品牌推广 | 品牌内容、产品介绍 | 1000-2000 字 | L4 案例嵌入 |

## v0.3.0 (2026-04-13)

### 新增功能
1. **企业多产品管理**
   - 支持一个企业对应多个产品
   - 在企业画像编辑页面新增产品管理功能
   - 可为每个产品单独设置描述和优势

2. **本地内容导入**
   - 支持本地 Markdown 文件一键导入
   - 支持 Markdown 内容直接粘贴导入
   - 快速分发到多平台

3. **自动获取出口 IP**
   - 系统自动获取本地出口 IP
   - 方便配置微信公众号等平台的 IP 白名单

4. **内容自动改编**
   - 多平台发布时内容智能适配
   - 新增"重写中"发布状态
   - 发布流程：等待中 → 重写中 → 发布中 → 成功/失败

5. **新增 51CTO 平台**
   - 支持平台数增至 13 个

### 文档更新
- 创建使用手册 V4.0
- 更新搜索引擎名称：阿里云 IQS 搜索
- 更新企业画像说明：一个企业对应多个产品
- 更新发布状态说明
- 使用中文图片名称
