# 版本更新记录

## v0.4.0 (2026-06-04)

### 新增功能
1. **数据回收 - GEO 排名检测**
   - 全面了解品牌在多个 AI 搜索引擎上的表现
   - 支持检测平台：百度 AI、DeepSeek、Kimi、豆包、腾讯元宝等
   - 助力优化 GEO（生成式引擎优化）策略
   - 实时监控品牌在 AI 搜索结果中的排名变化

---

## v0.3.0 (2026-04-13)

### 新增功能
1. **企业多产品管理**
   - 支持一个企业对应多个产品
   - 在企业画像编辑页面新增产品管理功能
   - 可为每个产品单独设置描述和优势

2. **本地内容导入**
   - 支持本地 Markdown 文件一键导入
   - 支持 Markdown 内容直接粘贴导入
   - 快速分发到多平台

3. **自动获取出口 IP**
   - 系统自动获取本地出口 IP
   - 方便配置微信公众号等平台的 IP 白名单

4. **内容自动改编**
   - 多平台发布时内容智能适配
   - 新增"重写中"发布状态
   - 发布流程：等待中 → 重写中 → 发布中 → 成功/失败

5. **新增 51CTO 平台**
   - 支持平台数增至 13 个

### 文档更新
- 创建使用手册 V4.0
- 更新搜索引擎名称：阿里云 IQS 搜索
- 更新企业画像说明：一个企业对应多个产品
- 更新发布状态说明
- 使用中文图片名称
