# Focus GEO

> AI 驱动的内容生成与多平台分发工具，让内容营销效率提升 10 倍

Focus GEO 是一款专为内容营销团队设计的 AI 工具，帮助企业快速生成高质量的营销内容，并一键发布到 13 个主流自媒体平台。

---

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🤖 **AI 内容生成** | 基于企业画像和关键词，10 分钟生成一篇专业文章 |
| 🔍 **关键词蒸馏** | AI 蒸馏 + 自动组合，批量生成高质量长尾关键词 |
| 📚 **知识库管理** | 集中管理文档、图片资源，AI 参考生成更精准 |
| 🚀 **多平台发布** | 一键发布到 13 个平台，内容自动改编适配 |
| 📊 **GEO 排名检测** | 检测品牌在百度 AI、DeepSeek、Kimi、豆包、腾讯元宝等 AI 搜索引擎的表现 |
| 📝 **本地导入** | 支持 Markdown 文件一键导入和分发 |
| 💰 **积分系统** | 灵活的积分充值与功能解锁机制 |

## 🚀 快速开始

### 第一步：扫码登录

进入系统后，点击右下角【登录】按钮，使用微信扫码登录。

### 第二步：完成基础配置

进入【基础配置】页面，使用一键配置 API 或手动配置大模型和搜索引擎。

### 第三步：创建企业画像

在【企业画像】中填写企业信息，支持一个企业对应多个产品。

### 第四步：生成内容

使用【关键词蒸馏】生成关键词，然后在【AI 内容生成】中批量生成文章。

### 第五步：发布内容

进入【多平台发布】，将内容一键分发到各个平台。

## ❓ 常见问题

### API Key 设置后提示无效？

- 检查 API Key 是否输入正确，有无多余空格
- 确认 API Key 有足够余额和调用额度
- 检查 Base URL 是否填写正确

### 如何配置微信公众号发布？

1. 登录 [微信开发者平台](https://developers.weixin.qq.com/platform)
2. 获取 AppID 和 AppSecret
3. 添加 IP 白名单（系统可自动获取出口 IP）

### 生成的内容质量不理想？

- 完善企业画像信息，尤其是产品描述和优势
- 上传相关行业资料到知识库
- 调整生成参数，尝试不同的文章类型和风格
