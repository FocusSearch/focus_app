## 产品简介

### 什么是 Focus GEO？

Focus GEO 是一款专为内容营销团队设计的 AI 工具，通过以下核心能力帮助企业提升内容生产效率：

- **智能内容生成**：基于企业画像和关键词，自动生成优质文章
- **关键词挖掘**：AI 蒸馏和自动组合生成高质量长尾关键词
- **知识库管理**：集中管理文档、图片和标签资源
- **多平台分发**：一键发布到微信公众号、知乎、今日头条等 13 个平台
- **GEO 排名检测**：检测品牌在百度 AI、DeepSeek、Kimi、豆包、腾讯元宝等 AI 搜索引擎的表现
- **积分系统**：灵活的积分充值与功能解锁机制
- **内容自动改编**：多平台内容智能适配

![](https://github.com/FocusSearch/focus_app/blob/main/images/系统总览.png)

### 适用场景

- 企业公众号日常运营
- 产品营销推广内容制作
- SEO 优化文章批量生成
- 多平台内容同步分发
- 本地 Markdown 内容一键分发

### 核心优势

| 特性 | 说明 |
|------|------|
| 高效 | 10 分钟生成一篇专业文章 |
| 精准 | 基于企业画像生成定制化内容 |

| 灵活 | 积分制按需解锁功能 |
| 自动 | 内容自动改编，多平台适配 |

## 快速开始

### 第一步：扫码登录

**登录后才能开始基础配置**。进入系统后，点击右下角【登录】按钮，使用微信扫码登录。

![](https://github.com/FocusSearch/focus_app/blob/main/images/扫码登录.png)

### 第二步：完成基础配置

进入【基础配置】页面，配置大模型 API 和搜索引擎 API。

#### 一键配置 API（推荐）

扫码登录后，使用官方提供的 token 快速配置官方提供的基础模型和搜索引擎：

1. 进入【全局配置】
2. 【引擎来源】选择「App Center 默认引擎」（选择后自动保存配置）
3. 点击【测试链接】验证
4. 点击【保存】完成配置

![](https://github.com/FocusSearch/focus_app/blob/main/images/一键配置api.png)

### 第三步：创建企业画像

在【企业画像】中填写您的企业信息，这是 AI 生成内容的基础。

### 第四步：添加产品

在【企业画像】→【编辑】→【产品管理】中为您的企业添加多个产品。

### 第五步：生成关键词

使用【关键词蒸馏】功能生成与您业务相关的长尾关键词。

### 第六步：生成内容

在【AI 内容生成】中选择产品画像和关键词，批量生成文章。

### 第七步：发布内容

进入【多平台发布】，将内容一键分发到各个平台。

## 应用中心与积分

应用中心是 Focus GEO 的核心管理系统，提供积分充值、功能解锁、邀请奖励等功能。

进入方式：左下角【应用中心】

### 积分获取

#### 1. 充值

用户通过 **微信支付** 或 **支付宝** 发起充值。

**基础汇率**：¥1 = 100 积分

**阶梯奖励倍率**：充值金额越高，获得积分越多。

| 充值金额（¥） | 奖励倍率 | 示例（充值该金额获得积分） |
|-------------|---------|----------------------|
| < 100       | 1.0×    | ¥50 → 5,000 积分     |
| ≥ 100       | 1.1×    | ¥100 → 11,000 积分   |
| ≥ 300       | 1.2×    | ¥300 → 36,000 积分   |
| ≥ 500       | 1.3×    | ¥500 → 65,000 积分   |

**计算公式**：`积分 = 充值金额 × 汇率 × 阶梯倍率`

**充值限制**：
- 最低充值：¥10
- 单笔上限：¥100,000

#### 2. 邀请新用户

每成功邀请一位新用户注册，获得 **1,000 积分**。

**操作步骤**：
1. 进入【应用中心】→【个人信息】
2. 复制推广链接
3. 分享链接邀请他人注册
4. 用户成功注册后自动获得奖励

#### 3. 新用户奖励

新用户注册后自动获得 **1,000 积分**（默认配置）。

#### 4. 首充奖励

用户首次充值成功后，系统自动赠送关联的功能套餐（如"技术社区套餐"）。

### 积分消耗

| 消耗类型 | 费用 | 说明 |
|---------|------|------|
| 搜索消耗 | 10 积分/次 | 单次搜索 |
| LLM 输入 | 1,000 积分/百万 Token | 大模型输入消耗 |
| LLM 输出 | 1,500 积分/百万 Token | 大模型输出消耗 |
| LLM 缓存 | 100 积分/百万 Token | 缓存 Token 消耗 |

### 功能解锁

进入方式：【应用中心】→【应用功能】

![](https://github.com/FocusSearch/focus_app/blob/main/images/功能解锁.png)

![](https://github.com/FocusSearch/focus_app/blob/main/images/应用功能.png)

#### 功能分类

- **免费功能**：无需解锁，所有用户直接可用
  - 微信、知乎、头条等平台

- **付费功能**：需消耗积分解锁后才能使用

#### 解锁方式

##### 1. 单个功能解锁

直接选择某个付费功能，用积分解锁。

##### 2. 套餐解锁

购买预定义的功能套餐，打包更优惠。

**普通套餐**：

| 套餐名称 | 包含功能 | 原价 | 折扣价 | 折扣 |
|---------|---------|------|-------|------|
| 技术社区套餐 | CSDN、掘金、博客园、思否、51CTO | 6,250 | 4,000 | 64% |
| 新闻资讯套餐 | 相关新闻平台 | 3,000 | 2,000 | 67% |
| 视频套餐 | B站 | 2,000 | - | - |
| 云服务套餐 | 腾讯云 | 2,000 | - | - |
| 优质媒体套餐 | InfoQ | 3,000 | - | - |

##### 3. 全功能通行证

一次性解锁全部 13 个功能。

| 套餐名称 | 包含功能 | 原价 | 折扣价 |
|---------|---------|------|-------|
| 全功能通行证 | 全部 13 个功能 | 13,250 | 9,000 |

**特殊定价规则**：已购买的单个功能/套餐花费可抵扣。

```
实际费用 = MAX(0, 通行证价格 - 已消费购买积分总额)
```

**示例**：用户之前单独购买功能花了 5,000 积分，通行证价 9,000，则只需再付 4,000 积分。

#### 解锁来源

| 来源 | 说明 | 是否可撤销 |
|------|------|-----------|
| 用户购买 | 用户积分购买 | 不可撤销 |
| 管理员授予 | 管理员手动授予 | 可由管理员撤销 |
| 促销赠送 | 促销活动赠送 | 不可撤销 |

## 基础配置

### 大模型设置

Focus GEO 支持 OpenAI 兼容的 API 接口，可对接各类大语言模型服务。

![](https://github.com/FocusSearch/focus_app/blob/main/images/大模型设置.png)

#### 配置步骤

1. **Base URL** - 输入 API 服务地址
   - 示例：`https://api.openai.com/v1` 或您的自定义 API 地址
   - 支持主流服务商：OpenAI、Azure、Anthropic、国内大模型平台等

2. **API Key** - 输入您的 API 密钥
   - API Key 将**加密存储**，保存后显示为掩码（如 `sk-****abcd`）
   - 请确保密钥具有足够的调用额度

3. **模型 ID** - 指定要使用的模型标识符
   - 示例：`gpt-4`、`gpt-3.5-turbo`、`claude-3-sonnet` 等
   - 请根据您的 API 服务商填写正确的模型名称

#### 💡 配置建议

| 使用场景 | 推荐模型 | 说明 |
|---------|---------|------|
| 日常内容生成 | GPT-3.5 / 同级模型 | 性价比高，响应快 |
| 深度长文创作 | GPT-4 / Claude-3 | 质量更高，逻辑更强 |
| 中文内容优化 | 国产大模型 | 中文表达更自然 |

### 搜索模型设置

Focus GEO 集成多个搜索引擎，用于实时信息检索和内容参考。

![](https://github.com/FocusSearch/focus_app/blob/main/images/网络搜索设置.png)

#### 支持的搜索引擎

| 搜索引擎 | 特点 | 适用场景 |
|---------|------|---------|
| **阿里云 IQS 搜索** | 中文搜索结果精准 | 国内内容创作、热点追踪 |
| **百度 AI 搜索** | 全网信息覆盖广 | 综合信息检索、资料收集 |

#### 配置步骤

1. **搜索引擎类型** - 选择您要使用的搜索引擎
   - 下拉选择：阿里云 IQS / 百度 AI 搜索

2. **API Key** - 输入搜索引擎的 API 密钥
   - 请从对应服务商控制台获取
   - API Key 将**加密存储**，保存后显示为掩码

3. **返回结果数** - 设置每次搜索返回的结果数量
   - 建议：5-10 条
   - 数量越多，参考资料越丰富，但生成时间会增加

4. **超时时间 (ms)** - 搜索请求超时时间
   - 单位：毫秒
   - 建议：5000-10000ms（5-10秒）
   - 网络较慢时可适当增加

#### 🔧 获取 API Key

##### **阿里云 IQS：**
1. 登录阿里云控制台
2. 进入「人工智能」→「智能搜索」
3. 创建应用并获取 API Key

##### **百度 AI 搜索：**
1. 注册百度智能云账号
   - 访问 [https://cloud.baidu.com/](https://console.bce.baidu.com/qianfan/overview)
   - 完成 **实名认证**（个人即可）

2. 进入千帆大模型平台
   - 控制台地址：[https://console.bce.baidu.com/qianfan](https://link.zhihu.com/?target=https%3A//console.bce.baidu.com/qianfan)

3. 创建 API Key

> 注意：这里用的是 **API Key（Bearer Token）**，不是 AK/SK！

操作路径：
```
左侧菜单 → 【系统管理】→【API Key】→ 【创建 API Key】
```

- **权限策略**：默认即可（已包含 AI 搜索权限）
- 复制生成的 Key（格式如 `bce-v3/ALTAK-xxxxx`）

## 企业画像与产品管理

企业画像是 Focus GEO 的核心基础，AI 根据这些信息生成符合您企业定位的内容。**一个企业可管理多个产品**，支持企业多产品管理模式。

![](https://github.com/FocusSearch/focus_app/blob/main/images/企业画像.png)

1. 点击【新建画像】按钮
2. 填写企业基础信息（带 * 为必填项）
3. 点击【保存】
4. 在【编辑】→【产品管理】中添加多个产品

---

### 基础信息

| 字段 | 限制 | 说明 |
|------|------|------|
| **画像名称** * | 最多 20 字 | 用于区分不同画像，如"企业版产品"、"个人版产品" |
| **公司全称** * | 最多 50 字 | 营业执照上的正式名称 |

| **公司网站** | URL 格式 | 官网链接，用于 AI 了解企业 |
| **补充画像** | 最多 1000 字 | 其他需要 AI 知道的背景信息 |

## 关键词蒸馏

关键词蒸馏帮助您挖掘高质量的长尾关键词，这些关键词是 AI 生成内容的基础。Focus GEO 提供两种生成方式：**AI 蒸馏**和**自动组合**。

---

### AI 蒸馏

AI 根据您提供的主关键词，自动生成符合搜索意图的长尾关键词。

![](https://github.com/FocusSearch/focus_app/blob/main/images/Pasted image 20260305145340.png)

#### 配置参数

| 参数 | 说明 | 示例 |
|------|------|------|
| **主关键词** * | 核心业务词，最多 100 字 | "AI 写作工具"、"数据分析软件" |
| **生成数量** | 需要生成的关键词数量 | 建议 10-50 个 |
| **生成方式** | 选择生成策略 | 精准模式 / 扩展模式 |
| **问题语言** | 目标用户语言 | 中文 / 英文 |
| **问题类型** | 关键词的意图类型 | 可多选 |

#### 问题类型说明

## GEO 排名检测

> v0.4.0 新增功能

GEO（Generative Engine Optimization，生成式引擎优化）排名检测功能，帮助您全面了解品牌在各大 AI 搜索引擎上的表现，优化您的 GEO 策略。

### 什么是 GEO？

GEO 是针对 AI 搜索引擎的优化策略，与传统 SEO 不同，GEO 关注的是品牌在 AI 生成回答中的曝光度和排名。

### 支持的 AI 搜索引擎

| 平台 | 说明 |
|------|------|
| **百度 AI 搜索** | 百度旗下的 AI 搜索功能 |
| **DeepSeek** | 国内领先的 AI 搜索引擎 |
| **Kimi** | 月之暗面推出的 AI 助手 |
| **豆包** | 字节跳动推出的 AI 助手 |
| **腾讯元宝** | 腾讯推出的 AI 助手 |

### 功能说明

#### 排名检测

- 输入品牌关键词或产品关键词
- 选择要检测的 AI 搜索引擎平台
- 系统自动查询并分析品牌在各平台的表现
- 生成详细的排名报告

#### 检测维度

| 维度 | 说明 |
|------|------|
| **品牌提及率** | 品牌在 AI 回答中被提及的频率 |
| **排名位置** | 品牌在推荐列表中的位置 |
| **内容关联度** | 品牌与搜索关键词的相关性 |
| **竞品对比** | 与竞争对手的排名对比 |

### 使用场景

1. **品牌监控** - 定期检测品牌在 AI 搜索中的曝光情况
2. **竞品分析** - 了解竞争对手在 AI 搜索中的表现
3. **策略优化** - 根据检测结果调整内容营销策略
4. **效果评估** - 评估 GEO 优化措施的效果

### 操作步骤

1. 进入【数据回收】→【GEO 排名检测】
2. 输入要检测的关键词（品牌词、产品词等）
3. 选择要检测的 AI 搜索引擎平台
4. 点击【开始检测】
5. 查看检测结果和分析报告
6. 根据报告优化您的内容策略

### 最佳实践

- **定期检测**：建议每周至少检测一次，跟踪排名变化
- **多关键词覆盖**：检测品牌词、产品词、行业词等多种关键词
- **竞品对比**：同时检测竞品关键词，了解市场竞争格局
- **结合内容优化**：根据检测结果调整关键词策略和内容方向

### Q10：发布到微信公众号失败？

**配置步骤：**
1. **访问新平台并登录**：
    
    - 打开 [微信开发者平台](https://developers.weixin.qq.com/platform) (https://developers.weixin.qq.com/platform)。
        
    - 点击右上角，使用公众号管理员的微信扫码登录。
        
2. **进入“我的业务”**：
    
    - 登录后，在左侧菜单栏找到并点击 **“我的业务”**。
        
    - 在业务列表中，选择你的 **公众号** 或 **服务号**，点击进入管理页面。![[微信开发者平台.png]]
        
3. **找到并配置所需信息**：
    
    - 在管理页面中，你主要会用到“基础信息”和“开发密钥”两个区域。
        
        - **`AppID`**：在 **“基础信息”** 区域可以直接看到。
            
        - **`AppSecret` & IP白名单**：点击 **“开发密钥”**，这里可以管理 `AppSecret`（启用/重置）和 **配置 `API IP白名单`**![[获取appid&appsecret.png]]


**排查步骤：**
1. 检查公众号的 AppID 和 AppSecret 是否正确
2. 确认是否已添加 IP 白名单（可使用系统自动获取的出口 IP）
3. 确认公众号已开通【原创】功能（如需要）
4. 检查文章是否包含敏感词
5. 查看微信公众平台的违规通知


### Q11：知乎平台发布失败？

**排查步骤：**
1. 知乎平台仅支持**扫码登录**，请确保使用正确方式授权
2. 检查账号认证状态是否为【已认证】
3. 确认账号是否有**发文权限**
4. 检查文章是否包含敏感词
5. 查看平台的违规通知
